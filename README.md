# Ubiquitous-Computing
Coursework Specification: The Invisible Caregiver

The system implements a complete smart home data acquisition, storage and processing
pipeline, leveraging edge computing technologies and AI. The system ingests data from sensors,
manages their storage and semantic organisation, and exposes the data to third-party
applications and external users as required. 

The elderly person is in good physical health, but must take some medication regularly. As most
elderly persons, the home occupant may be a little forgetful, showing mild signs of dementia. The
system must be able to support their basic activities and routines, such as cooking, washing
themselves, engaging in some social or leisure activities, allowing them to live well
independently.

The system collects data from the sensors and stores them in a small home server. The data is
accessible to an edge AI device (Jetson Nano) that is able to analyse it and produce a natural
language response, summarising arising issues, or answering questions of remote care-takers
about the home occupant’s wellbeing, through a chat interface.
